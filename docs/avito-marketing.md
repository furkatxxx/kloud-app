# База знаний: Маркетинг на Авито

> Последнее обновление: 2026-02-22
> Статус: ЗАПОЛНЕНО (все основные секции)

## Алгоритм ранжирования Авито

Авито использует **ML-модель** (машинное обучение), которая предсказывает вероятность конверсии — насколько вероятно, что пользователь совершит целевое действие (звонок, сообщение, просмотр контактов).

### Факторы ранжирования (по приоритету)

| Фактор | Влияние | Что делать |
|--------|---------|-----------|
| CTR объявления (клики/показы) | Очень высокое | Качественное первое фото + точный заголовок + конкурентная цена |
| Конверсия в контакты | Очень высокое | Описание должно отвечать на все вопросы, снижать барьер к звонку |
| Свежесть (дата публикации) | Высокое | Поднимать в пиковые часы (10-12 и 18-21) |
| Геолокация | Высокое | Указывать точный адрес |
| Полнота карточки | Среднее | Заполнить ВСЕ атрибуты категории |
| Рейтинг продавца | Среднее | Собирать отзывы |
| Скорость ответа | Среднее | Отвечать в течение 1 часа → метка "быстро отвечает" (+15-20% CTR) |
| История продаж | Среднее | Завершать сделки через Авито |

### Что поднимает позицию
- Высокий CTR на превью
- Много контактов относительно просмотров
- Свежая дата публикации
- Быстрые ответы на сообщения
- Положительные отзывы
- Полностью заполненная карточка

### Что опускает позицию
- Низкий CTR (видят, но не кликают)
- Жалобы на объявление
- Игнорирование сообщений
- Старая дата без обновлений
- Неполная карточка
- Дубликаты объявлений

### Редактирование и позиция
- Редактирование **НЕ** меняет дату публикации, **НЕ** поднимает в поиске
- Манипуляции ценой (туда-обратно) **НЕ** работают, могут привести к пессимизации
- Редактирование полезно для улучшения карточки (фото, описание) → улучшает CTR

### Штрафы
| Нарушение | Последствие |
|-----------|-------------|
| Дубликаты (одно и то же в одном городе) | Скрытие дубля, предупреждение |
| Спам в заголовке (!!!, CAPS, лишние символы) | Пессимизация или отклонение |
| Несоответствие категории | Пессимизация, перемещение |
| Жалобы пользователей | Скрытие, проверка модератором |
| Подозрительные паттерны | Блокировка поднятия |
| Слишком низкая цена | Флаг "мошенничество", проверка |

### Платные услуги НЕ влияют на органику
- Платное продвижение = отдельные рекламные слоты НАД органикой
- После окончания платного периода → возврат на органическую позицию
- Платный трафик и органика — разные системы

## Платные инструменты продвижения

### Список инструментов

| Инструмент | Что делает | Для B2B |
|------------|-----------|---------|
| **Поднятие** | Разово поднимает в верх выдачи на несколько часов | Полезно 1-2 раза в неделю (вт, чт утром) |
| **XL-объявление** | Увеличенная карточка в выдаче, больше текста и фото видно | Рекомендуется — создаёт солидность |
| **Выделение цветом** | Жёлтый фон заголовка | Слабый эффект для B2B |
| **Турбопродажа** | Пакет: поднятие + XL + выделение + доп. показы (7 дней) | Только при обновлении ассортимента |
| **Быстрая продажа** | Максимальное продвижение, топ + рекомендации | Дорого, не для длительных B2B циклов |
| **Настройка цены просмотра (бид)** | Аукцион: платишь за каждый просмотр | Основной инструмент для B2B |

### Настройка цены просмотра (бид) — подробно
- Авито предлагает "рекомендуемый бид" — стартуй с него +20-30%
- Ставить ниже рекомендованного бесполезно — будешь в хвосте
- Обязательно ставь **дневной лимит** расходов
- Считай не CPV (стоимость просмотра), а **CPL (стоимость лида)**
- Формула: если 100 просмотров → 3 обращения → 1 покупка, то CPL = бюджет/1

### Автостратегии
| Стратегия | Когда использовать |
|-----------|-------------------|
| Ручное управление | На старте (первые 2-4 недели) для контроля |
| Максимум показов | Когда есть статистика и нужен охват |
| Цель — обращения | Предпочтительна для B2B — оптимизирует конверсию, а не показы |

### Стратегия для дорогого B2B товара (100+ тыс. руб.)
1. **Присутствие, а не взрывы** — постоянный бид топ-5 на 30 дней лучше Турбопродажи на 3 дня
2. **XL-формат на весь период** — доверие через визуальную солидность
3. **Поднятие 1-2 раза в неделю** (вт/чт утром) — всплеск в горячие часы
4. **Расписание показов** — отключать в выходные и ночью (экономия 30-40%)
5. **Быстрый ответ** (15-30 мин) — каждый лид на вес золота

### Оптимальное время для B2B
- Лучшие дни: вторник — четверг
- Лучшие часы: 9:00–12:00, 13:00–16:00 МСК
- Избегать: пятница после 15:00, выходные, праздники

### Примерный бюджет на тест (1 месяц)
- XL-формат 30 дней: ~3-8 тыс. руб.
- Бид с дневным лимитом 200-500 руб. (рабочие дни): ~4-10 тыс. руб.
- 4-8 поднятий: ~500-2000 руб.
- **Итого: 8-20 тыс. руб./мес** — если 1 сделка на 100+ тыс., ROI отличный

## Оптимизация объявлений

### Заголовки
- **Максимум 50 символов**, в мобильной выдаче видно ~30-35 → самое важное в начало
- Формула для B2B: `[Бренд/Модель] + [Ключевой параметр] + [Условие]`
- Пример: `Bitmain Antminer T21 190TH НДС ГТД` (36 символов)
- Слова повышающие CTR для B2B: **НДС**, **ГТД**, **опт**, **партия**, **с документами**, конкретные цифры (TH/s)
- **Запрещено**: CAPS LOCK, телефоны, email, `!!!`, `***`, слова «лучший/самый дешёвый»
- Всегда указывать полную модель + хешрейт — покупатели ищут по точной модели

### Описания
- Минимум 500-800 символов, оптимум **2000-3000** для дорогого B2B
- Максимум Авито: ~5000 символов
- Ключевые слова в описании **влияют на поиск** (заголовок + описание индексируются)
- Не спамить — повторение одного слова 5+ раз → штраф
- Использовать синонимы: майнер, асик, ASIC, bitcoin miner, криптооборудование

**Структура описания (по блокам):**
1. **Лид** (первые 150-200 символов — видны до "Читать далее"): главный оффер, НДС, ГТД, количество
2. **Техническая таблица**: модель, хешрейт, потребление, алгоритм, состояние, год
3. **Условия сделки**: документы (счёт-фактура, УПД, ГТД), безнал, договор поставки, минимальная партия, доставка
4. **Доверие**: опыт, объём реализации, легальный ввоз
5. **Призыв к действию**: "Пишите в чат — ответим в течение часа. Подготовим КП"

**Что повышает конверсию в контакт:**
- Конкретные цифры > общие слова ("75 единиц в наличии" > "большая партия")
- Явный формат работы: "работаем с юрлицами по договору поставки"
- Снятие возражений: "НДС выделен, ГТД предоставляем"
- Обещание скорости: "КП в течение 2 часов"
- Предложение осмотра на складе — мощный конвертер для сделок от 3-5 млн

### Фотографии
- Минимум 5-8, оптимум **10-15** для дорогого B2B (Авито допускает до 20)
- Первое фото = обложка в поиске: устройство целиком, нейтральный фон, горизонтальное (4:3)
- **Без водяных знаков** с контактами — Авито блокирует и штрафует

**Что снимать для B2B:**
| Фото | Зачем |
|------|-------|
| Оборудование на нейтральном фоне | Обложка, CTR |
| Паллеты/коробки на складе | Масштаб партии |
| Шильдик/серийный номер | Доверие, проверяемость |
| Документы (ГТД) без конфиденциальных данных | Снимает главный страх |
| Склад с оборудованием | Реальный бизнес |
| Крупный план разъёмов | Техническая аудитория |
| Экран с хешрейтом (тест) | Работоспособность |

**Технические требования:**
- Формат: JPG, PNG
- Минимум: 640x480, рекомендуемо: 1200x900+
- Макс. размер: 25 МБ на фото

### Цена
- Указывать **цену за единицу**, не за партию — попадает в больший диапазон фильтров
- В описании: "Цена за 1 шт. при партии от 30 шт. Цена партии — по запросу"
- Не ставить "договорная" — такие объявления показываются хуже
- Не писать "торг" в заголовке — снижает ценность для B2B
- Слишком низкая цена → проверка на фрод
- Резкое снижение НЕ даёт буста, манипуляции ценой → пессимизация

### Категория и параметры
- Майнеры размещать: **Электроника → Компьютерное оборудование → Серверы и сетевое**
- Альтернатива: Оборудование и производство → Промышленное оборудование
- Проверять, где больше конкурентов по нужной модели — туда и идти (там трафик)
- **Заполнять ВСЕ доступные параметры** — прямо влияет на ранжирование
- Объявление с полными параметрами попадает в фильтрованную выдачу

## B2B продажи на Авито

### Позиционирование
- Явно писать: **"Работаем с юридическими лицами"**
- Указывать формы оплаты: безнал, договор поставки, счёт-фактура
- Упоминать сроки закрытия сделки — юрлица планируют бюджеты
- Если возможна рассрочка/отсрочка — сильный конвертер

### Доверие для B2B
- **Верифицированный аккаунт** (галочка) — обязательно
- **Бизнес-аккаунт** (шильдик "Бизнес") — повышает доверие юрлиц
- Давность аккаунта имеет значение — новый аккаунт = минус
- Отзывы: просить первых клиентов оставить отзыв
- Сайт компании — указывать в профиле (в объявлении ссылки блокируются)

### Партионные продажи
- Упоминать конкретный остаток: "75 единиц на складе в Москве"
- Фото конкретной партии, не рендеры
- Предлагать осмотр на складе — конвертер для сделок 3-5+ млн
- Указывать логистику: "Доставка ТК по РФ, самовывоз со склада"

## Аналитика и метрики
- Показы — сколько раз объявление появилось в выдаче
- Просмотры — сколько раз открыли карточку
- Контакты — показали телефон / написали
- Конверсия показы→просмотры = качество превью (фото+заголовок+цена)
- Конверсия просмотры→контакты = качество карточки (описание, фото, доверие)
- Избранное — интерес, но не готовность купить
- Место в поиске — позиция среди конкурентов

## Бизнес 360

### Что это
Комплексное решение для профессиональных продавцов на Авито. Включает:
- Расширенная аналитика (просмотры, контакты, конверсии по каждому объявлению)
- Управление несколькими объявлениями из единого кабинета
- Шильдик "Бизнес" на объявлениях → повышает доверие
- Доступ к рекламным форматам
- Выделенный менеджер (на старших тарифах)
- Подключение CRM и API

### Как подключить
1. Зайти на `avito.ru/avito-business` или `pro.avito.ru`
2. Нажать "Подключить для бизнеса"
3. Указать ИНН, данные компании
4. Выбрать тариф (зависит от категории и региона)
5. Для крупных клиентов — запросить персонального менеджера

### Для нас
Бизнес-аккаунт **обязателен** для продаж дорогого B2B-оборудования. Шильдик "Бизнес" напрямую влияет на конверсию юрлиц.
