## ПОСЛЕ ВСЕХ ПРАВОК — САМОПРОВЕРКА

По каждому объявлению открой его и проверь:
1. Заголовок содержит: модель + хешрейт + НДС + ГТД + партия
2. Цена в карточке = цена в описании
3. Количество в карточке = количество в описании
4. Описание НЕ обрезано, НЕ зациклено, читается до конца
5. Нет "аптайм 65%" нигде
6. Все три описания содержат: "безналичный расчёт", "НДС", "ГТД РФ", "юридическое лицо"
